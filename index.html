Prêmios Incríveis

Explore os Desafios

1. PokeData Explorer

Descrição: Usando a PokéAPI, desenvolva uma aplicação que apresente informações de forma interativa sobre o universo Pokémon.

Objetivo: Criar uma solução que envolva exploração de dados, visualização e interação (Ex: Battle Simulator, Quiz, Comparador).

2. Meu Bairro Digital

Descrição: Use alguma API pública de dados abertos (como o dados.gov.br) para criar uma solução que melhore a vida urbana.

Objetivo: Transformar dados públicos em informações úteis para o cidadão (Ex: Painel de saúde, App de feedback, Visualizador de estatísticas).

3. Desafio Criativo Livre

Descrição: Os participantes podem escolher qualquer API pública (ou combinar várias) para resolver um problema real.

Objetivo: Avaliar criatividade, integração técnica e apresentação da solução (Ex: Combinar Spotify + Wikipedia, APIs de geolocalização).
